# TeamD_FinalProject

いさんぽ JAPAN

## 開発環境起動

docker compose up --build

## 停止

docker compose down

##

http://localhost:3000 → frontend が表示される
http://localhost:8000/docs → backend FastAPI の Swagger UI が表示される
http://localhost:8080 → pgAdmin ログイン可能

## 環境構築済み

Next.js TypeScript
FastAPI Python
ORM SQLAlchemy
DB PostgreSQL
DB 管理 pgAdmin
Lint ESLint
Format Prettier
